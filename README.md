# 화곡1동 민간 도심복합사업 안내 사이트

주민 여러분께 사업 구역과 동의 절차를 안내하는 정적 웹페이지입니다.

## 구성

- `index.html` — 메인 페이지 (구역도 / 동의서 안내 / 추진위 연락처)
- `style.css` — 스타일시트
- `assets/` — 이미지 리소스 (구역도 등)
- `notion-import.md` — 노션 임포트용 마크다운 (선택적 대안)

## 로컬 확인

```bash
# 브라우저로 직접 열기
start index.html

# 또는 로컬 서버로 (Python 설치 시)
python -m http.server 8000
# → http://localhost:8000
```

## 배포

Cloudflare Pages 또는 GitHub Pages 등 정적 호스팅에 그대로 업로드 가능합니다.
